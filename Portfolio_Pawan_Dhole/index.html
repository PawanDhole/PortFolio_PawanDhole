2.Product Management System

The Product Management System is a web-based application designed to handle the lifecycle of product management in a business environment. Developed using Java Spring Boot for the backend and MySQL for database management, this system provides a robust solution for managing products, including their creation, modification, deletion, and retrieval. The application also includes comprehensive API testing with Postman to ensure the reliability and correctness of its RESTful endpoints.

Key Features

1. Product Management
  • Create Products:Add new products to the system with details such as name, description, price, and category.
  • Read Products: Retrieve and view the list of products with options to filter by various criteria (e.g., category, price range).
  • Update Products: Modify existing product details to reflect changes in the inventory or pricing.
  • Delete Products: Remove products from the system when they are no longer available or needed.

Technologies Used:

  • Java Spring Boot
  • MySQL
  • JPA/Hibernate
  • Spring Security
  • Postman

Note: This project involves multiple microservices and may not have a single live demo.
